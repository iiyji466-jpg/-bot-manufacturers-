# 🤖 بوت صانع البوتات (Bot Factory)

بوت تيليجرام يمكّنك من إنشاء بوتات احترافية مخصصة دون كتابة أي كود برمجي.

## ✨ المميزات

- 🎯 **قوالب جاهزة**: اختر من بين عدة قوالب:
  - بوت أوامر وردود بسيط
  - بوت تحميل من يوتيوب وانستغرام
  - بوت ترجمة فورية
  - بوت محادثة ذكي (ChatGPT)
- 📦 **حزمة نشر كاملة**: يصدر البوت ملف ZIP يحتوي على كود البوت الجديد، ملف المتطلبات، وتعليمات النشر.
- 🚀 **استضافة مستقلة**: كل بوت يتم إنشاؤه يمكن نشره على حساب Render خاص بالمستخدم ليعمل 24/7 دون توقف.
- 🔒 **آمن**: لا يتم تخزين توكنات البوتات أو مفاتيح API بعد انتهاء الجلسة.

## 🛠️ طريقة الاستخدام

1. ابدأ محادثة مع البوت وأرسل `/start`.
2. اختر "🤖 صنع بوت جديد".
3. اختر نوع البوت من القائمة.
4. أدخل توكن البوت الجديد (من @BotFather).
5. أدخل رسالة الترحيب.
6. (إذا لزم الأمر) أدخل مفاتيح API المطلوبة.
7. راجع الملخص ثم اضغط "🚀 أنشئ البوت!".
8. ستصلك حزمة ZIP تحتوي على كل ما تحتاجه لنشر البوت على Render.

## 📤 نشر البوت المستلم

- فك ضغط الملف واتبع التعليمات في ملف `README.md` المرفق.
- ستحتاج إلى حساب مجاني على [Render.com](https://render.com).
- بعد النشر، استخدم [UptimeRobot](https://uptimerobot.com) لمنع البوت من الدخول في وضع السكون.

## 🖥️ تشغيل البوت الرئيسي (للمطورين)

### المتطلبات
- Python 3.10+
- تثبيت المكتبات: `pip install -r requirements.txt`
- الحصول على توكن بوت من @BotFather

### التشغيل المحلي
```bash
export BOT_TOKEN="توكن_البوت_الخاص_بك"
python bot_factory.py
